# GCTUIModalPresentationViewController
- - - - - 
* 快速使用的集成多种动画方案的模态视图控制器。
* TextView\TextField 同步键盘模态弹出视图支持。

## 功能
- [ ] 支持13种 `presentAnimat`
## 安装

#### 支持手动导入。
直接导入工程中 `GCTUIModalPresentationViewController` 文件，继承即可使用。

#### 支持 `Cocoapods` 导入

```ruby
pod 'GCTUIModalPresentationViewController'
```
